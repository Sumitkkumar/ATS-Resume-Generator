# ATS Resume Generator 

Generate highly optimized, ATS-friendly resumes using AI tailored to any Job Description.

---

## Features

- JD URL scraping
- ATS keyword optimization
- Grey-hat skill expansion (optional)
- Quantified experience & project bullets
- Clean professional PDF output
- Dark-themed modern UI
- One-click PDF download

---

## Tech Stack

### Frontend
- React + Vite
- Dark UI (ChatGPT-inspired)
- Responsive design

### Backend
- Python + FastAPI
- Gemini LLM
- PDF generation (FPDF)
- Intelligent JD parsing

### Infrastructure
- Docker
- Docker Compose
- Nginx
